# 小说生成器 — 使用手册

## 这是啥

你输入灵感或片段，AI帮你写好一整章（约5000字），去掉AI味，保证前后不矛盾。每天四章，发平台。

---

## 第一次用：新建一部小说

```
/novel-init
```

AI会问你几个问题：
- 小说叫什么名字
- 主角是谁（名字、性格、大概经历）
- 世界是什么样的（有没有修炼、门派之类）
- 大概想写一个什么故事

不需要想清楚所有细节，**能说多少说多少**，后面随时可以补。

---

## 日常写作：写新章节

```
/novel-write
```

然后告诉AI：
- 什么都不说 → AI根据大纲自己决定写什么
- 说灵感 → "这章主角要发现师父的秘密，情绪很激动"
- 贴片段 → 把你写的开头贴进去，AI续写到5000字

AI会自动：
1. 读取之前发生的所有情节
2. 写草稿
3. 自己评分、自己改（最多改2轮）
4. 去掉AI腔
5. 输出成品 + 更新情节记录

---

## 规划接下来写什么

```
/novel-arc
```

当你不知道接下来10章写什么时用。AI会给你一个粗大纲，你可以改，改完存起来作为后续写作的参考。

---

## 目录结构

```
novels/
  你的小说名/
    人物本子.md      ← 记着所有重要人物
    世界规则.md      ← 修炼、门派、世界设定
    大纲.md          ← 主线走向
    情节流水账.md    ← AI自动记录已发生的事
    我的文风.md      ← 你的写作偏好
    chapters/        ← 所有已写章节
```

---

## 常见问题

**Q：人物设定写错了怎么办？**
直接编辑 `人物本子.md`，AI下次写作会读新版本。

**Q：AI写的不对，情节走偏了？**
编辑 `情节流水账.md`，删掉或修改有问题的那条记录，再重新 `/novel-write`。

**Q：想换一种风格？**
编辑 `我的文风.md`，写上你想要的或不想要的。

**Q：章节输出在哪里？**
`novels/你的小说名/chapters/` 目录里，文件名是 `ch001_章节名.md` 这样的格式。
